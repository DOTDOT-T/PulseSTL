# Pulse Standard Template Library
The PSTL is a library of containers made with C++23, with algorithms, iterators and tools for runtime and development.
The objectif of the PSTL is to be complete, and easily usable for all of our project at Pulse Software.

## How to use
By copying the repo, you can generate example with CMake and the main.cpp available but also compile the library to a usable DLL.
